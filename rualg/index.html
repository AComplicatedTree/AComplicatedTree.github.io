RUALG

Info

RuAlg is a cubing bot that displays algorithms in a discord text channel. RuAlg uses the best algs from speedcubedb and displays them along with the images and an alternative algorithm. It also shows a video link with the proper fingertricks so you can get it right!

Note: RuAlg currently has the pll algset from cfop but more algsets from different methods are to be added in a few months.

Getting Started

To get started, invite RuAlg to your server. Type / in a text channel. Click on @RuAlg's profile picture. You will see a list of all the available commands. Detailed explanataion on how to use every command is listed in the next section.

Commands

Command Name Command Usage Command Info
help /help Returns the help command which shows a list of all the commands.
pll /pll [pll_name] Returns the pll alg requested.
Example Usage: /pll jb
ping /ping Returns the bot response time.
about /about Returns basic information about the bot.

About The Bot

RuAlg is developed by @AComplicatedTree#4773 using the Discord.JS library and is hosted on heroku. The idea of making a bot with such functionality came to me after a month or two of joining discord. I wanted to make an algorithm bot really badly but I lacked the knowledge of code. I spent a lot of time and made a bot with bdfd but I wasn't satisfied with it, so I gave up. A year after the idea struck me and here we are with the public version of RuAlg. The original version of RuAlg had OLL and COLL algsets as well since it was created during my annual holidays. Adding more algsets will definitely take more time and is not something I can do until the end of March. I have more ideas other than algs planned that will start development after OLL and COLL gets added.
